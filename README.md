# Detalk

> :tada: Detalk is a simple comment plugin.


**If you want to add Detalk to your website, the fastest way is [@detalk/static](https://github.com/oCoke/detalk-static).**

## HTTP API

### Add Comment

```
PUT /_api/comment
```

### Get Comment

```
GET /_api/comment
```

### Delete Comment

```
DELETE /_api/comment
```
